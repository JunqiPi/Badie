# Product Overview

**羽毛球搭子 (BadmintonBuddy)** - iOS social matching app for finding badminton partners.

## Core Value Proposition
Connect badminton players based on skill level and location for singles (1v1) or doubles (2v2) matches.

## Key Features
- **Splash**: Animated launch screen with badminton bounce animation
- **Auth**: Login/registration with skill level selection
- **Home**: Map view showing nearby players + game mode selection
- **Matching**: Real-time player matching with pulse search animation
- **Match Success**: Card collision effect with particle explosion
- **Profile**: User stats (games played, win rate)
- **Confirmation**: Match booking details

## User Flow
Splash → Auth → Home → Select Mode → Matching → Match Success → Confirm

## Target Users
Badminton enthusiasts in China looking for local playing partners.

## Localization
Primary language: Simplified Chinese (zh-CN)
